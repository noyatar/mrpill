## הוראות שימוש

### מבנה התיקיות
כל מוצר כולל:
- `README.md` - תיאור מפורט של המוצר
- `images/` - תמונות המוצר (1-4 תמונות)
- `prompts/` - פרומפטים ליצירת סרטונים (יתווסף)
- `content/` - תוכן מוכן לפרסום (יתווסף)

### הוספת מוצר חדש
1. צור תיקייה חדשה: `mkdir "שם-מוצר-חדש/images"`
2. הורד תמונות לתיקיית images
3. צור README.md בהתבסס על הדוגמה
4. עדכן את הטבלה הזו

### סקריפטים זמינים
- `../scripts/download_all_images.sh` - הורדת תמונות מעודכנות
- `../scripts/batch_download.sh` - הורדה קצרה
- `../scripts/create_readme.sh` - יצירת README (יתווסף)
