# 💊 מיסטר פיל - מערכת ניהול תוכן

מערכת מסודרת לניהול תוכן, תמונות ופרומפטים עבור מוצרי מיסטר פיל

## 📁 מבנה הפרויקט

```
mrpill/
├── products/           # כל המוצרים
│   ├── מקלילון-לרציניים/
│   │   ├── images/    # תמונות המוצר
│   │   └── README.md  # תיאור המוצר
│   ├── נובטמין-לגיימרים/
│   ├── מנחוסון-למנחוסים/
│   └── ...
├── scripts/           # סקריפטים לאוטומציה
│   └── download_product_images.sh
├── content/           # תוכן מוכן לפרסום (יתווסף)
├── prompts/           # פרומפטים לסרטונים (יתווסף)
└── README.md
```

## 🛍️ המוצרים

**סה"כ 14 מוצרים זמינים:**
- מקלילון לרציניים
- נובטמין לגיימרים  
- מנחוסון למנחוסים
- סאחינול לסאחי
- טיולין למטיילים
- ביישנול לביישניים
- חרפן לעייפים
- פרפקטמול לפרפקציוניסטים
- חפרלין לחופרים
- דרמטין לדרמטיות
- עצלנומצין לעצלנים
- הפתעה איחורון למאחרים
- גיוס צהובון לחיילים
- סטלנומי לסטלנים

## 🔧 שימוש

### הורדת תמונות מוצרים
```bash
cd scripts
./download_product_images.sh
```

### הוספת מוצר חדש
1. צור תיקיה חדשה ב-`products/`
2. צור תת-תיקיה `images/`
3. הוסף `README.md` עם תיאור המוצר
4. עדכן את `products/product_list.md`
