## 📁 מבנה הפרויקט

```
mrpill/
├── products/                    # כל המוצרים (20)
│   ├── מקלילון-לרציניים/
│   │   ├── images/             # תמונות המוצר (1-4)
│   │   └── README.md          # תיאור מפורט
│   ├── נובטמין-לגיימרים/      # 4 תמונות איכותיות
│   ├── מנחוסון-למנחוסים/
│   ├── סאחינול-לסאחי/         # 4 תמונות

│   ├── download_all_images.sh  # הורדת תמונות מעודכנות
│   └── batch_download.sh       # הורדה מהירה

└── README.md                   # המדריך הזה
```

## 🔧 איך להשתמש

### הורדת תמונות מעודכנות
```bash
cd scripts
./download_all_images.sh
```

### הוספת מוצר חדש
```bash
# צור תיקיה חדשה
mkdir "products/מוצר-חדש/images"

# הוסף תמונות והכן README
# עדכן את products/product_list.md
```

### בדיקת סטטוס מוצרים
```bash
cd products
for dir in */; do
  echo "$dir: $(ls $dir/images/ 2>/dev/null | wc -l) תמונות"
done
```
